Obstruction Viewer For Starlink

If using Starlink router, please go to your STARLINK Debug Panel: , after a few seconds, you will see code. Click anywhere in the area where you see the code, hit CTRL key + A. This will select the code in blue. Then hit CTRL+C. This copies all the information. Come back to this page, click in the white box on the left and paste (CTRL+V) your code into the box. A graph will appear if you have obstructions. If not, you are good!

Credit for creating this viewer goes to u/ChuckTSI, who originally shared it in a reddit thread. It is rehosted here with modifications to demonstrate questions.
